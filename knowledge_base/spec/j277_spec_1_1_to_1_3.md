
# OCR GCSE Computer Science (J277)
# Component 01 – Computer Systems
## Units 1.1 – 1.3

---

# 1.1 Systems architecture

## 1.1.1 Architecture of the CPU

Students should be able to:

- Describe the architecture of the Central Processing Unit (CPU):
  - The Von Neumann architecture:
    - MAR (Memory Address Register)
    - MDR (Memory Data Register)
    - Program Counter
    - Accumulator
    - Control Unit
    - ALU (Arithmetic Logic Unit)
    - Cache
    - Registers
  - The role of registers
  - The fetch-decode-execute cycle
  - Common CPU components and their function

---

## 1.1.2 CPU performance

Students should be able to:

- Describe the factors affecting the performance of the CPU:
  - Clock speed
  - Cache size
  - Number of cores

- Describe how the CPU executes instructions sequentially

- Explain the impact of:
  - Embedded systems
  - Modern CPU architecture on performance

---

## 1.1.3 Embedded systems

Students should be able to:

- Explain what is meant by an embedded system
- Describe examples of embedded systems
- Explain the advantages and disadvantages of embedded systems

---

# 1.2 Memory and storage

## 1.2.1 Primary storage (memory)

Students should be able to:

- Describe the function of the CPU and how it interacts with memory
- Explain the difference between RAM and ROM
- Describe the purpose of RAM and ROM
- Describe the differences between volatile and non-volatile memory
- Explain the purpose of virtual memory

---

## 1.2.2 Secondary storage

Students should be able to:

- Describe the need for secondary storage
- Describe common types of storage:
  - Optical
  - Magnetic
  - Solid state

- Compare the advantages and disadvantages of different storage types
- Describe suitable storage devices and storage media for a given application

---

# 1.3 Computer networks, connections and protocols

## 1.3.1 Networks and topologies

Students should be able to:

- Define:
  - LAN (Local Area Network)
  - WAN (Wide Area Network)

- Describe:
  - Star topology
  - Bus topology
  - Mesh topology

- Explain:
  - The role of a router
  - The role of a switch
  - The role of a wireless access point
  - The role of a NIC (Network Interface Controller)

- Explain the concept of:
  - Client-server networks
  - Peer-to-peer networks

---

## 1.3.2 Wired and wireless networks, protocols and layers

Students should be able to:

- Describe:
  - Ethernet
  - Wi-Fi
  - Bluetooth

- Describe common network protocols:
  - TCP/IP
  - HTTP
  - HTTPS
  - FTP
  - SMTP
  - POP3
  - IMAP

- Explain the concept of layers:
  - TCP/IP model
  - The purpose of layers

- Explain packet switching

- Explain IP addressing and MAC addressing

- Explain DNS (Domain Name Server)
